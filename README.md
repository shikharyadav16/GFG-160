## 🚀 How to Run

1.  Clone this repository:

    ``` bash
    git clone https://github.com/shikharyadav16/GFG-160.git
    ```

2.  Open the repo in VS Code:

    ``` bash
    cd GFG-160
    code .
    ```

3.  Run any solution file:

    ``` bash
    python Strings/01_str_to_int_atoi.py
    ```
